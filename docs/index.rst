Welcome to |project|'s documentation
====================================

|project| is an extension for Flask that which adds support for quickly crating wehbooks for Google Assistant App.

Get started with :doc:`installation` and then get an overview with the :doc:`quickstart`.

|project| depends on Flask. The documentation for this library can be found at

- `Flask documentation <https://flask.palletsprojects.com/en/2.0.x/>`_
